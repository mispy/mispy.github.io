Resources

I spend way too much time on the internet, and I tend to encounter a lot of interesting and useful stuff. It seems like I should really be keeping a record of what said stuff is and why it is useful!

Free Software

  • dotPeek is a .NET assembly decompiler. It produces something very close to readable C#, even from complex programs like Unity games.
  • SQLiteStudio is a Windows program useful for inspecting SQLite databases.

Non-Free Software

  • ReSharper is an excellent plugin for Visual Studio which does much more sophisticated static analysis.

Social Statistics

  • Bradd Libby has an excellent aggregation of data sources for various subjects.
  • Equaldex is a collaborative knowledge base which tracks LGBT legal rights around the world.
  • 2014 Essential Facts About the Computer and Video Game Industry illustrates that gaming demographics are unlike what people expect, with women at 48% and rising quickly.
  • Google's Public Data Directory
  • The Global Terrorism Index 2015 goes into a lot of depth wrt. violent extremism, particularly the socioeconomic and political causation
