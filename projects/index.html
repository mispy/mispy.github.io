Projects

Stuff I've made or helped make. Levels of seriousness and silliness may vary wildly!

Collaborative

  • SciRate - http://github.com/scirate/scirate

Ranking and commenting system for arXiv papers which I maintain. Originally created by Dave Bacon and the quant-ph community, who are the primary users. A Rails site, using a combination of postgres and elasticsearch for the storage backend. I gave a talk about SciRate at an open science workshop.

  • JSIL - https://github.com/sq/JSIL

JSIL is a C# to JavaScript transpiler, primarily used for porting games to the web. Katelyn contracted me to help work on getting JSIL to compile FNA, the end result of which was the ability to use P/Invoke to call Emscripten-compiled code. That is, you can compile C/C++ native libraries to JavaScript and then call them from C# also compiled to JavaScript. It's about as mad as it sounds!

  • Wander

A very pretty non-competitive multiplayer exploration game, written in C++/Lua and based on CRYENGINE. Loki hired me to help port this from PC to PS4, and I ended up delving into a lot of game networking concepts in the process.

Libraries

  • twitter_ebooks - https://github.com/mispy/twitter_ebooks

A Ruby framework for making interactive twitterbots. My bots play up the ELIZA effect by applying keyword statistics, simulated delays and other strategies for mimicing humans. The results are often very amusing.

  • libtcod-ruby - https://github.com/mispy/libtcod

Ruby FFI bindings for libtcod, a library which provides a bunch of tools like fast tile-based pathfinding and lighting for creating old-school roguelike games.

  • arxivsync - https://github.com/mispy/arxivsync

Ruby OAI interface for harvesting the arXiv, which powers SciRate. Can be used to store, update and parse a full mirror of the paper metadata in XML.

Chrome Extensions

  • DeviantART Collector - https://github.com/mispy/devicollect

Chrome extension which checks for new deviations and opens them all in individual tabs. I think this is actually the best software I've ever written, because it does exactly one little thing and does it well, and I've used it all the time for years.

  • Dotsies Chrome - https://github.com/mispy/dotsies_chrome

Chrome extension for learning the Dotsies alphabet, which is the typograhical equivalent of a conlang. Much easier to install and toggle than changing your entire system font.

Toys & Experiments

  • spdgrid - https://github.com/mispy/spdgrid

An HTML5 implementation of the prisoner's dilemma applied to a spatial grid, based on a paper by Nowak and May. I also added a mode which allows each cell to build up score across generations, which creates really alien-looking emergent structures.

Game Mods

  • Mechatronics - https://github.com/mispy/Mechatronics

[needs update] A mod for RimWorld which adds friendly constructable versions of the game's rogue mechanoid enemies. RimWorld's modding system is interesting in that it involves writing C# and hooking dlls straight into the existing game code, instead of operating in a sandbox with a limited scripting language.

  • Civil War Followers

A mod for Skyrim which makes the Imperial Legion and Stormcloak titles more useful by allowing you to order your poor subordinates about. The Elder Scrolls series has a really amazing modding community and it was nice to give something back to them!
